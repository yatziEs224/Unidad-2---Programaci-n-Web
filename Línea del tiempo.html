Programación Web

1950

FORTRAN

Primer lenguaje de programación.

1987

PERL

Aparece primer lenguaje de programación para desarrollo de páginas web.

1989-1995

PHP y HTML

Se crea el lenguaje de programacón PHP así como el Lenguaje de marcas de hipertexto HTML.

1996

JavaScript

Creación del lenguaje más interactivo, JavaScript. Creación del primer correo electrónico.

2005

YouTube

Se lanza la plataforma de YouTube, páginas para subir y ver videos.

2008-2013

HTML 5

Se crea HTML versión 5. Se crea la plataforma Git. Se crea Nodo Js.

2015

Visual Studio Code

Aparece el editor de texto Visual Studio Code.

2016-2019

HTTPS y CSS

Se crea HTTPS, protocolo seguro de transferencia de hipertexto. Se crea CSS, Hojas de estilo en cascada.
